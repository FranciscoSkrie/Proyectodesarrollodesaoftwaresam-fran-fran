v5 - Ajustes de roles, publicaciones y gestión

Cambios y arreglos:
- Eliminado el módulo de Monedero en menú, cuenta y navegación.
- Unificados los roles Cliente/Vendedor en un único rol Usuario.
- Corregida la navegación de “volver atrás”.
- Ajustadas las alertas de sesión para desaparecer automáticamente a los 3 segundos.
- Modificada la vista de publicaciones para quitar lógica de cupos y compra interna.
- Agregada función de cambio de contraseña en cuenta.
- Ajustado el panel de gestión para administrar foros/publicaciones.
- Mejorado el honeypot para registrar IP, usuario y datos del navegador.
- Ocultados detalles técnicos de bloqueo en “Mis publicaciones”.
- Actualizada transferencia de entradas para usar ID de usuario.
